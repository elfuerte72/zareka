**Терминальный ИИ-агент для написания кода — целиком на русском языке и целиком на российских моделях.** Собственный TUI на [OpenTUI](https://github.com/anomalyco/opentui) (React) поверх агентного ядра [opencode](https://github.com/anomalyco/opencode). При запуске играет гимн. Отключается гимн, разумеется, клавишей Esc — но зачем?

> Статус: художественно-инженерный эксперимент. Работает по-настоящему: агент читает файлы, пишет код, запускает команды. Фидбек и issues приветствуются, доносы — в спам.

## Постановление о возможностях

- **Полностью отечественные модели.** YandexGPT 5.1 Pro через [Yandex AI Studio](https://yandex.cloud/): OpenAI-совместимый API, оплата в рублях, данные не покидают территорию. Никаких VPN, зарубежных карт и — с этого релиза — никаких иностранных моделей. Tool calling проверен: модель на вопрос «работает?» ответила «трудится».
- **Два режима труда, переключение Shift+Tab:**
  - ☭ **ГОСПЛАН** — режим планирования: агент изучает и предлагает план, ничего не трогая;
  - ★ **ИСПОЛКОМ** — режим исполнения: агент претворяет план в жизнь.
- **Команды в соответствии с номенклатурой:**

  | Команда | Западный аналог | Назначение |
  |---|---|---|
  | `/открыть-дело` | `/new` | новая сессия |
  | `/чистка` | `/clear` | очистить экран |
  | `/пятилетка` | `/init` | изучить проект и составить AGENTS.md |
  | `/распределение` | `/model` | выбрать модель |
  | `/главк` | `/provider` | выбрать провайдера |
  | `/помощь` | `/help` | справка |
  | `/выход` | `/exit` | покинуть пост |

  Западные аналоги работают тоже — переучиваться не заставляем.
- **Гимн при запуске** — торжественное открытие каждой смены (кладёте свой mp3, см. ниже).
- **Санкции на действия** — когда агент хочет выполнить рискованную команду, запрашивается санкция: `1` — разрешить, `2` — на всю сессию, `3` — отказать.
- **Закрытый контур** — внешние MCP-серверы из ваших прочих конфигов отключаются автоматически: меньше поверхность утечки, а заодо не превышается лимит Яндекса в 100 инструментов на запрос.
- **Русский всё** — интерфейс, ответы агента, комментарии к коду, сообщения коммитов.

Агентное ядро — опенсорсный opencode, запускается подпроцессом; его логику Наркомпрог не переписывает, а получает и обновляет от апстрима. Своё здесь — интерфейс, русская адаптация и идеологическая надстройка.

## Ввод в эксплуатацию

### 1. Установите opencode и Bun

```bash
curl -fsSL https://opencode.ai/install | bash
curl -fsSL https://bun.sh/install | bash
```

### 2. Получите доступ к Yandex AI Studio

1. Зарегистрируйтесь в [Yandex Cloud](https://yandex.cloud/) и создайте каталог (folder).
2. Создайте API-ключ для сервисного аккаунта с ролью `ai.languageModels.user`.
3. Запомните **идентификатор каталога** (folder ID, вида `b1g...`).

### 3. Установите Наркомпрог

```bash
git clone https://github.com/elfuerte72/zareka && cd zareka
./install.sh   # пропишет провайдера Яндекса и русский промпт-пак в конфиг opencode
bun install
```

### 4. (По желанию) добавьте гимн

Аудио в репозиторий не входит (права на запись). Положите свой mp3 в корень проекта под именем `anthem-of-the-ussr-classical.mp3` — или укажите путь через `ZAREKA_ANTHEM`. Без файла смена открывается тихо.

### 5. Приступайте

```bash
export YANDEX_API_KEY="ваш-ключ"   # или добавьте в ~/.zshrc
bun start                          # TUI в каталоге вашего проекта
bun run smoke                      # самопроверка связки без TUI
```

## Клавиши

| Клавиша | Действие |
|---|---|
| `Enter` | отправить запрос |
| `Shift+Tab` | переключить режим: ГОСПЛАН ↔ ИСПОЛКОМ |
| `/` | команды (автоподсказка по мере набора) |
| `Esc` | выключить гимн · прервать агента · закрыть пикер · отказать в санкции |
| `1`/`2`/`3` | ответ на запрос санкции |
| `Ctrl+C` | выход |

## Переменные окружения

| Переменная | Назначение |
|---|---|
| `YANDEX_API_KEY` | ключ Yandex AI Studio |
| `ZAREKA_ANTHEM` | путь к mp3 гимна |
| `ZAREKA_NO_ANTHEM=1` | тихий запуск |
| `ZAREKA_PORT` | порт локального opencode-сервера (по умолчанию 4996) |

## Известные ограничения

- Компонент `<markdown>` текущей версии OpenTUI прозу не рендерит — Наркомпрог выводит текст сам, а код-блоки через `<code>` с подсветкой (Tree-sitter). Вернём полный markdown, когда починят апстрим.
- GigaChat в `/главк` пока «ожидает постановления» — адаптер (OAuth с обновлением токена каждые 30 минут + сертификат Минцифры) следующий в пятилетке.
- Диффы, история сессий — в работе.

## Наименования

**Наркомпрог** — Народный комиссариат программирования: так представляется агент. Кодовое имя репозитория — **zareka** («место за рекой»): инструмент для тех, кто работает по эту сторону, на своей инфраструктуре и своём языке.
