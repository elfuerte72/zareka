# Зарека

**Русскоязычный кодинг-агент в терминале.** Надстройка над [opencode](https://github.com/anomalyco/opencode): агент говорит по-русски и работает на моделях, доступных из России с оплатой в рублях, — из коробки.

> Статус: эксперимент. Работает, развивается по мере интереса. Фидбек и issues очень приветствуются.

## Что даёт Зарека

- **Пресет Yandex AI Studio** — Qwen3 235B (контекст 262K) через OpenAI-совместимый API. Оплата в рублях, данные не покидают РФ, никаких VPN и зарубежных карт.
- **Русский промпт-пак** — агент объясняет, комментирует и пишет коммиты по-русски; технические термины и идентификаторы остаются в оригинале.
- Дальше по плану: плагин GigaChat (OAuth + сертификат Минцифры), пресеты других доступных из РФ провайдеров, русский TUI.

## Быстрый старт

### 1. Установите opencode

```bash
curl -fsSL https://opencode.ai/install | bash
# или: npm install -g opencode-ai
```

### 2. Получите доступ к Yandex AI Studio

1. Зарегистрируйтесь в [Yandex Cloud](https://yandex.cloud/) и создайте каталог (folder).
2. Создайте API-ключ для сервисного аккаунта с ролью `ai.languageModels.user`.
3. Запомните **идентификатор каталога** (folder ID, вида `b1g...`).

### 3. Установите Зареку

```bash
git clone https://github.com/penkin/zareka && cd zareka
./install.sh
```

Установщик спросит API-ключ и folder ID, запишет конфиг в `~/.config/opencode/` (существующий конфиг бережно сохраняется в бэкап) и подключит русский промпт-пак.

### 4. Запустите

```bash
export YANDEX_API_KEY="ваш-ключ"   # или добавьте в ~/.zshrc
opencode
```

### Ручная установка

Не хотите запускать скрипт — скопируйте [`presets/yandex-ai-studio.opencode.json`](presets/yandex-ai-studio.opencode.json) в `~/.config/opencode/opencode.json`, замените `{FOLDER_ID}` на свой folder ID и положите [`prompts/zareka.ru.md`](prompts/zareka.ru.md) в `~/.config/opencode/zareka.ru.md`.

## Другие модели

- **DeepSeek / Qwen напрямую** — opencode поддерживает их штатно (`opencode auth login`), если у вас есть способ оплаты.
- **GigaChat, YandexGPT** — в планах как опции; следите за issues.

## Известные ограничения

- Поддержка tool calling у OpenAI-совместимого endpoint Yandex для open-source моделей проверяется — если агент спотыкается на вызовах инструментов, заведите issue с логом.
- Русский пока только у агента (промпты), интерфейс TUI — английский. Полностью русский TUI — следующий этап (форк с автопереводом).

## Почему «Зарека»

Зарека — «место за рекой»: инструмент для тех, кто работает по эту сторону, на своей инфраструктуре и своём языке.
