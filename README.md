# Зарека

**Русскоязычный кодинг-агент в терминале — с собственным интерфейсом.** Своё TUI-приложение на [OpenTUI](https://github.com/anomalyco/opentui) поверх агентного ядра [opencode](https://github.com/anomalyco/opencode) (то же ядро, что у официального клиента, — через `opencode serve`). Агент говорит по-русски и работает на моделях, доступных из России с оплатой в рублях, — из коробки.

> Статус: эксперимент. Работает, развивается по мере интереса. Фидбек и issues очень приветствуются.

## Что даёт Зарека

- **Собственный русский TUI** — интерфейс целиком на русском: лента чата, статусы, подписи действий агента («→ читаю файл», «✓ выполняю команду»).
- **Пресет Yandex AI Studio** — Qwen3 235B (контекст 262K) через OpenAI-совместимый API. Оплата в рублях, данные не покидают РФ, никаких VPN и зарубежных карт. Tool calling проверен — агентный цикл работает.
- **Русский промпт-пак** — агент объясняет, комментирует и пишет коммиты по-русски; технические термины и идентификаторы остаются в оригинале.
- Дальше по плану: разрешения на действия из TUI, история сессий, плагин GigaChat (OAuth + сертификат Минцифры), пресеты других доступных из РФ провайдеров.

└─────────────────────┘                └──────────┬───────────┘
                                                  │ OpenAI-совместимый API

Ядро обновляется апстримом opencode само — Зарека развивает только интерфейс и русскую адаптацию.

## Быстрый старт

### 1. Установите opencode и Bun

```bash
curl -fsSL https://opencode.ai/install | bash
curl -fsSL https://bun.sh/install | bash
```

### 2. Получите доступ к Yandex AI Studio

1. Зарегистрируйтесь в [Yandex Cloud](https://yandex.cloud/) и создайте каталог (folder).
2. Создайте API-ключ для сервисного аккаунта с ролью `ai.languageModels.user`.
3. Запомните **идентификатор каталога** (folder ID, вида `b1g...`).

### 3. Установите Зареку

```bash
git clone https://github.com/elfuerte72/zareka && cd zareka
./install.sh      # пропишет провайдера Яндекса и русский промпт-пак в конфиг opencode
bun install
```

Установщик спросит folder ID, запишет конфиг в `~/.config/opencode/` (существующий конфиг бережно сохраняется в бэкап) и подключит русский промпт-пак.

### 4. Запустите

```bash
export YANDEX_API_KEY="ваш-ключ"   # или добавьте в ~/.zshrc
bun start                          # TUI Зареки в каталоге вашего проекта
bun run smoke                      # быстрая самопроверка связки без TUI
```

Классический opencode при этом тоже продолжает работать с этим конфигом — командой `opencode`.

### Ручная установка

Не хотите запускать скрипт — скопируйте [`presets/yandex-ai-studio.opencode.json`](presets/yandex-ai-studio.opencode.json) в `~/.config/opencode/opencode.json`, замените `{FOLDER_ID}` на свой folder ID и положите [`prompts/zareka.ru.md`](prompts/zareka.ru.md) в `~/.config/opencode/zareka.ru.md`.

## Другие модели

- **DeepSeek / Qwen напрямую** — opencode поддерживает их штатно (`opencode auth login`), если у вас есть способ оплаты.
- **GigaChat, YandexGPT** — в планах как опции; следите за issues.

## Известные ограничения

- TUI Зареки — ранняя версия: чат, стрим ответа и действия агента. Разрешения на опасные действия, история сессий и выбор модели из интерфейса — в работе.
- Если агент спотыкается на вызовах инструментов — заведите issue с логом.

## Почему «Зарека»

Зарека — «место за рекой»: инструмент для тех, кто работает по эту сторону, на своей инфраструктуре и своём языке.
