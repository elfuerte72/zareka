# Зарека ☭

**Русскоязычный кодинг-агент в терминале — со своим интерфейсом.** Собственное TUI-приложение на [OpenTUI](https://github.com/anomalyco/opentui) (React) поверх агентного ядра [opencode](https://github.com/anomalyco/opencode). Агент говорит по-русски, работает на моделях, доступных из России с оплатой в рублях, и одет в советскую эстетику — из коробки.

> Статус: эксперимент. Работает, развивается по мере интереса. Фидбек и issues очень приветствуются.

## Что даёт Зарека

- **Собственный русский TUI** — советская тема (красное знамя, золото серпа и молота ☭), лента чата, подсветка кода, карточки действий агента («✓ читаю файл», «✎ правлю файл»), спиннер с таймером, слэш-команды, модалка разрешений.
- **Гимн при запуске** — при старте TUI играет заданный трек; выключается клавишей **Esc** (и сам замолкает, когда вы отправляете первый запрос).
- **Пресет Yandex AI Studio** — Qwen3 235B (контекст 262K) через OpenAI-совместимый API. Оплата в рублях, данные не покидают РФ, никаких VPN и зарубежных карт. Tool calling проверен.
- **Русский промпт-пак** — агент объясняет, комментирует и пишет коммиты по-русски; технические термины и идентификаторы остаются в оригинале.

## Архитектура

```
┌────────────────────────┐   HTTP + SSE   ┌──────────────────────┐
│  Зарека (TUI)          │ ◄────────────► │  opencode serve      │
│  OpenTUI/React, рус.    │                │  агентное ядро, LLM, │
│  src/                  │                │  инструменты, права  │
└────────────────────────┘                └──────────┬───────────┘
                                                      │ OpenAI-совместимый API
                                           ┌──────────▼───────────┐
                                           │  Yandex AI Studio    │
                                           │  Qwen3 235B, рубли   │
                                           └──────────────────────┘
```

Ядро агента — это опенсорсный opencode (`opencode serve`), запускается подпроцессом. Логику агента Зарека **не переписывает** — берёт готовую и обновляет апстримом, а сама развивает интерфейс и русскую адаптацию.

## Быстрый старт

### 1. Установите opencode и Bun

```bash
curl -fsSL https://opencode.ai/install | bash
curl -fsSL https://bun.sh/install | bash
```

### 2. Получите доступ к Yandex AI Studio

1. Зарегистрируйтесь в [Yandex Cloud](https://yandex.cloud/) и создайте каталог (folder).
2. Создайте API-ключ для сервисного аккаунта с ролью `ai.languageModels.user`.
3. Запомните **идентификатор каталога** (folder ID, вида `b1g...`).

### 3. Установите Зареку

```bash
git clone https://github.com/elfuerte72/zareka && cd zareka
./install.sh   # пропишет провайдера Яндекса и русский промпт-пак в конфиг opencode
bun install
```

### 4. (Опционально) добавьте гимн

Аудио в репозиторий не входит (права на запись). Положите свой mp3 в корень проекта под именем `anthem-of-the-ussr-classical.mp3` — или укажите путь через переменную `ZAREKA_ANTHEM`. Без файла Зарека просто запускается молча.

### 5. Запустите

```bash
export YANDEX_API_KEY="ваш-ключ"   # или добавьте в ~/.zshrc
bun start                          # TUI Зареки в каталоге вашего проекта
bun run smoke                      # быстрая самопроверка связки без TUI
```

## Клавиши

| Клавиша | Действие |
|---|---|
| `Enter` | отправить запрос |
| `/` | слэш-команды (`/новая`, `/очистить`, `/помощь`, `/выход`) |
| `Esc` | выключить гимн · прервать ответ агента · отклонить разрешение |
| `1` / `2` / `3` | в модалке разрешений: один раз / на сессию / отклонить |
| `Ctrl+C` | выход |

## Переменные окружения

| Переменная | Назначение |
|---|---|
| `YANDEX_API_KEY` | ключ Yandex AI Studio |
| `ZAREKA_ANTHEM` | путь к mp3 гимна (по умолчанию — файл в корне проекта) |
| `ZAREKA_NO_ANTHEM=1` | не играть гимн |
| `ZAREKA_PORT` | порт локального opencode-сервера (по умолчанию 4996) |

## Другие модели

- **DeepSeek / Qwen напрямую** — opencode поддерживает их штатно (`opencode auth login`), если у вас есть способ оплаты.
- **GigaChat, YandexGPT** — в планах как опции; следите за issues.

## Известные ограничения

- Компонент `<markdown>` в текущей версии OpenTUI прозу не рендерит, поэтому Зарека выводит текст ответа сама (заголовки, списки, жирный), а код-блоки — через рабочий `<code>` с подсветкой синтаксиса. Полноценный markdown вернём, когда починят апстрим.
- TUI — ранняя версия. Диффы, история сессий и выбор модели из интерфейса — в работе.
